其它

  • 關閉輸入法整合器
  1. 開啟 C:\Windows\System32\IME\shared
  2. 找到 IMEPADSV.EXE
  3. 右鍵點擊内容 -> 安全頁 -> 進階 -> 更改擁有者為自己 -> 確定關閉
  4. 重新右鍵點擊内容 -> 安全頁 -> 編輯權限 -> 除了自己之外的其他賬戶權限都取消勾選
  5. 重開機
