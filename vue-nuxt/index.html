What is Nuxt.js？

Nuxt.js 是再以 Vue 為基礎所建構的框架，補足了 Vue.js 需要處理的 Server-Side Rendering 問題。

Nuxt.js 大概而言整合了這些套件：

  • Vue 本體

  • Vue Router：管理 Vue app 的路由

  • Vuex：實作單向資料流 flux 設計模式

  • Vue Server Renderer：處理 Server-Side Rendering 的 package

  • Vue-Meta：管理 Vue app 的 metadata

  • Learn Next.js - A free video course on React and Next.js.

安裝

npm install --save nuxt
npx create-nuxt-app nuxt_app
