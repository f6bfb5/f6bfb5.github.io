#### SubtitleEdit 常用快捷

| 按鍵    | 功能              |
| ------- | ----------------- |
| F8      | 播放/暫停影片     |
| F9      | 增加新字幕        |
| F11     | 設定字幕開始位置  |
| F12     | 設定字幕結束位置  |
| F5      | 播放被選取的字幕  |
| Alt+↑/↓ | 選取上/下一句字幕 |
