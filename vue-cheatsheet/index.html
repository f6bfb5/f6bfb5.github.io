Vue - Cheatsheet

August 22, 2020 3 min read

  • Vue-Essentials-Cheat-Sheet.pdf

Expressions

<div id="app">
  <p>I have a {{ product }}</p>
  <p>{{ product + 's' }}</p>
  <p>{{ isWorking ? 'YES' : 'NO' }}</p>
  <p>{{ product.getSalePrice() }}</p>
</div>

Directives

<p v-if="inStock">{{ product }}</p>
<p v-else-if="onSale">...</p>
<p v-else>...</p>

Toggles CSS property: display: none

<p v-show="showProductDetails">...</p>

Two-way data binding

<input v-model="firstName" />
  • v-model.lazy="..."
  • v-model.number="..."
  • v-model.trim="..."

List Rendering

<li v-for="item in items" :key="item.id">{{ item }}</li>

取得陣列座標：

<li v-for="(item, index) in items">...</li>

遞迴物件：

<li v-for="(value, key) in object">...</li>

使用於 component 時：

<cart-product
  v-for="iotem in products"
  :profuct="item"
  :key="item.id"
></cart-product>

Binding

<a v-bind:href="url">...</a>

可簡寫為

<a :href="url">...</a>
<button :disabled="isButtonDisabled">...</button>

當 isActive 為真時，會加入 active class

<div :class="{ active: isActive }">...</div>
<div :style="{ color: activeColor }"></div>

Actions / Events

<button v-on:click="addToCart">...</button>

可簡寫為

<button @click="addToCart">...</button>

可傳入參數

<button @click="addToCart(product)">...</button>

可預防如重載頁面的預設行為

<form @submit.prevent="addProduct">...</form>

或只觸發一次

<img @mouseover.once="showImage" />...
  • .stop：阻止所有 event propagation
  • .self：只有當 event.target 為元素本身時才觸發

按鍵觸發：

<input @keyup.enter="submit" />
<input @keyup.ctrl.c="onCopy" />

按鍵裝飾詞：

  • .tab
  • .delete
  • .esc
  • .space
  • .up
  • .down
  • .left
  • .right
  • .crtl
  • .alt
  • .shift
  • .meta

滑鼠裝飾詞：

.left .right .middle

Component Anatomy

Vue.component('my-component', {
  components: {
    // Components that can be used in template
    ProductComponent, ReviewComponent
  },
  props: {
    // The parameters the component accepts
    message: String,
    product: Object,
    email: {
      type: String,
      required: true,
      default: "none",
      validator: function (value) {
        // Should return true if value is vaild
      }
    }
  },
  data: function() {
    // must be a function
    return {
      firstName: 'Vue',
      lastName: 'Mastery'
    }
  },
  computed: {
    // Return cached values until dependcies change
    fullName: function() {
      return this.firstName + ' ' + this.lastName
    }
  },
  watch: {
    // called when firstName changes value
    firstName: function (value, oldValue) { ... }
  },
  methods: { ... },
  template: '<span>{{ message }}</span>,
  // Can also use backticks for multi-line
})

Custom Events

Use props (above) to pass data into child components, custom events to pass data to parent elements.

Set listener on component, within its parent:

<button-counter v-on:incrementBy="incWithVal"></button-counter>

Inside parent component:

methods: {
incWithVal: function (toAdd) { ... }
}

Inside button-counter template:

this.$emit("incrementBy", 5);

Using A Single Slot

Component template:

<div>
  <h2>I'm a title</h2>
  <slot> Only displayed if no slot content </slot>
</div>

Use of component with data for slot:

<my-component>
  <p>This will go in the slot</p>
</my-component>

Multiple Slots

Component template:

<div class="container">
  <header>
    <slot name="header"></slot>
  </header>
  <main>
    <slot>Default content</slot>
  </main>
  <footer>
    <slot name="footer"></slot>
  </footer>
</div>

Use of component with data for slot:

<app-layout>
  <h1 slot="header">Page Title</h1>
  <p>the main content.</p>
  <p slot="footer">Contact info</p>
</app-layout>
