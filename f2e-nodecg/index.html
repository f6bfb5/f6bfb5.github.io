使用 NODECG 妝點直播介面

November 13, 2019 1 min read

介紹

NodeCG 是為了直播所設計的介面框架，提供使用者能使用網頁撰寫動態與複雜的直播介面設計，透過操作快速修改介面內容。

範例：

  • SGDQ2018-Layouts
  • RTA in Japan-Layouts

安裝

前置需求

  • Node.js
  • npm (已於 Node.js 0.6.3 版後內建)
  • git

建立網站

git clone https://github.com/nodecg/nodecg.git
cd nodecg
npm install --production
node index.js

可於 cfg/nodecg.json 修改設定。

安裝 Bundle

  • awesome-nodecg
  1. 使用 nodecg-cli 安裝
npm install --global nodecg-cli
nodecg install lange/lange-notify
  1. 安裝來自 Bitbucket 的 Bundle

nodecg install bitbucket:username/repo-name

  1. 安裝來自其它 git 提供者的 Bundle

nodecg install https://gitlab.com/username/repo-name.git

自製 Bundle

  1. 安裝 Yeoman (網站樣板頁面工具) 和 generator-nodecg (快速產生 NodeCG 樣板檔案)
npm install --global yo generator-nodecg
cd bundles
mkdir my-first-bundle
cd my-first-bundle
yo nodecg

亦可指定產生特定類型的樣版頁面

yo nodecg:panel
yo nodecg:graphic
yo nodecg:extension
  1. 開啟 nodecg/bundles/bundle-sample/dashboard/panel.html 修改 Panel 頁面
  2. 開啟 nodecg/bundles/bundle-sample/extension.js 修改 JS
  3. 開啟 nodecg/bundles/bundle-sample/graphics/index.html 修改 Layout 頁面
